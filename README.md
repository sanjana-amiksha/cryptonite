# cryptonite
Cryptonite is a simple mobile android application that works to encrypt and decrypt textual data in step by step classical ciphers of Vigen√®re, Caesar and Affine. It was developed in Android Studio for version 33 and display tested on OnePlus 7 Pro updated. It is compatible with all Android phones' light mode. Only three ciphers are presently included. Information about each cipher is included in the application. It uses SQLite3 to create a cipher history.
